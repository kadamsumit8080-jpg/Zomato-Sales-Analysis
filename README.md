#  Zomato Restaurants Analysis Dashboard

An interactive *Power BI dashboard* built to analyze restaurant data across multiple countries using Zomato dataset insights.

---

##  Project Overview

This dashboard provides insights into:

- Total Restaurants
- Number of Cities
- Number of Countries
- Average Rating
- Total Votes
- Total Cuisines
- City-wise restaurant distribution
- Cuisine-wise performance
- Online delivery availability
- Table booking availability
